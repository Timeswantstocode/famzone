**FAM Zone** is a premium, full-stack media aggregation infrastructure — a unified dashboard that orchestrates **real-time sports analytics**, **multi-source metadata**, **AI-driven narrative reconstruction**, and **synchronized watch parties** into a cohesive, high-performance dark-themed interface. It is the public-facing counterpart to the private **FAM** project.

## 🔐 Security & Optimization

</div>

FAM Zone is engineered with a focus on network security and frontend performance:

> 🛡️ **Content Security Policy** — Strict CSP headers blocking inline scripts and unauthorized origins
>
> 🛡️ **HSTS** — Strict Transport Security for encrypted connections
>
> 🛡️ **X-Frame-Options** — Anti-clickjacking protection for the playback interface
>
> 🛡️ **X-Content-Type-Options** — MIME-type sniffing prevention
>
> 🛡️ **CORS Mitigation** — Custom server-side proxying + Cloudflare Workers for sensitive asset retrieval
>
> 🛡️ **Referrer-Policy** — Privacy-centric origin handling
>
> 🛡️ **Permissions-Policy** — Fine-grained hardware access control (camera, microphone, sensors)
>
> 🛡️ **Rate Limiting** — Upstash Redis-backed rate limiting with configurable tiers per route
>
> 🛡️ **Asset Caching** — Optimized ISR strategies and aggressive PWA caching via `@ducanh2912/next-pwa`
